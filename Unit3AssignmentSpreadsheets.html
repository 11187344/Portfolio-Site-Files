In Unit 9 Spreadsheets, we learned how to use spreadsheet software to organise, analyse, and present data effectively. The unit focused on helping us understand the basic and some advanced features of programs like Microsoft Excel.
We started by learning how to enter and format data, create formulas to perform calculations, and use functions like SUM, AVERAGE, and IF statements. It was interesting to see how spreadsheets can automatically update results when you change data, which makes working with numbers much easier and quicker. There was more function like IF that could make our results more accurate, but sometimes it would be long to write.
Through this unit, I developed practical skills in managing and organising data efficiently, solving a variety of problems using different formulas, and presenting information clearly through charts and graphs. Learning how to use functions like IF statements and VLOOKUP helped me understand how spreadsheets can automate complex tasks, saving time and reducing errors. I also gained experience in formatting spreadsheets to make them easy to read and professional-looking. Overall, I feel much more confident using spreadsheets now, and I can see how these skills are useful not only in lots of different jobs but also in everyday life, such as budgeting, planning events, or keeping track of personal information.
